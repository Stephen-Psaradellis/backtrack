## Card 1: Avatar-Based Matching

### Headline
**No Photos Needed**

### Subheadline
Describe who you saw with customizable avatars

### Description
Our unique avatar builder lets you describe someone using 11+ customizable features - hair color, style, skin tone, glasses, clothing, and more. No photos required means complete privacy for everyone.

### Visual Concept
- Split screen: Avatar builder UI on left, completed avatar on right
- Show attribute sliders/selectors
- Example avatar with varied features

### Key Points
- 11+ customizable attributes
- Primary features: skin, hair, facial hair
- Secondary features: eyes, clothes, accessories
- No photos = no privacy concerns

## Visual Design Guidelines for Feature Cards

### Layout Template

```
┌─────────────────────────────────────┐
│                                     │
│     [ICON or VISUAL ELEMENT]        │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  HEADLINE (Bold, 24-32pt)           │
│                                     │
│  Subheadline (Regular, 16-18pt)     │
│                                     │
│  Description text paragraph         │
│  with 2-3 sentences explaining      │
│  the feature in plain language.     │
│                                     │
│  • Key point 1                      │
│  • Key point 2                      │
│  • Key point 3                      │
│                                     │
└─────────────────────────────────────┘
```

### Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Primary (headlines, buttons) | Teal | #0D9488 |
| Secondary (accents, icons) | Amber | #F59E0B |
| Background | White | #FFFFFF |
| Background Alt | Light Gray | #F9FAFB |
| Text Primary | Dark Gray | #1F2937 |
| Text Secondary | Medium Gray | #6B7280 |

### Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headline | SF Pro / Roboto | Bold | 24-32pt |
| Subheadline | SF Pro / Roboto | Medium | 16-18pt |
| Body | SF Pro / Roboto | Regular | 14-16pt |
| Key Points | SF Pro / Roboto | Medium | 14pt |

### Icon Style
- Line icons (not filled)
- 2px stroke weight
- Teal primary color
- Consistent 24x24 or 32x32 size

## App Store Screenshot Order

Based on typical user attention, recommended screenshot sequence:

1. **Hero** - "Find who you noticed" with main value prop
2. **Avatar Builder** - Shows unique feature
3. **Location Discovery** - Real places, real connections
4. **Post View** - What a post looks like
5. **Chat** - The outcome - matching and chatting
6. **Privacy** - Safety and privacy features
